# Negocios Universales

> Plataforma AI de tasacion inmobiliaria y vehicular para el mercado bancario dominicano.

**Portal:** [app.negociosuniversales.ai](https://app.negociosuniversales.ai)

## Quick Start
```bash
cp .env.example .env.local
npm install
npm run dev
```

## Stack
- Next.js 16 + TypeScript + Tailwind
- Supabase (Auth + DB + Storage + pgvector)
- Claude AI (6 agentes pipeline)
- Vercel (deploy + edge)
